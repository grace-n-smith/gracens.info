Getting Started


Why GitHub?

GitHub is a free, cloud-based service that developers use to store and manage code. It has a super user-friendly interface, and has even been used for writing books. You will be using GitHub to connect a website (that you will edit locally) to a repository (repo) under your GitHub account. The repo will be identical to a folder local to your computer that stores the website's pages. GitHub will keep the files for the web pages that you create accessible on the cloud 24/7, without need for a personal server.
