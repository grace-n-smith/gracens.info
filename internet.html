Getting Started with GitHub


Why GitHub?

GitHub is a free, cloud-based service that developers use to store and manage code. It has a super user-friendly interface, and has even been used for writing books. You will be using GitHub to connect a website, that you will edit locally, to a repository (repo) under your GitHub account. The repo will be identical to a local folder on your computer that stores your website's pages. (this project will not utilize GitHub's full capacity for version control and collaboration, but it's a comfy place to start, even without coding experience)


What to install

  • A text editor
  • GitHub Desktop
